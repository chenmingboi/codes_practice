限制输出精度

```cpp
//方法1 使用ostringstream
ostringstream oss;
oss << fixed << setprecision(1) << output; //保留小数点后一位小数
//oss << setprecision(1) << output; //不指定fixed的时候，是有效数字位数
cout << oss.str();

//方法2 使用printf
printf("%.1f", output);//小数点后1位
printf("%.4g", output);//有效位数为4

//方法3 使用cout
cout << fixed << setprecision(1) << output;//保留小数点后一位
cout << setprecision(1) << output;//保留有效位数为一位
// 但是使用cout会有缺点，就是当上面的cout限制精度之后会影响下面的cout的精度，即使是下面的cout不设置精度限制
```

质数的数学性质

**试除法找到 n 的最小质因数，用 n 除以它即得最大质因数。**

原题：已知正整数 *n* 是两个不同的质数的乘积，试求出两者中较大的那个质数。

为什么第一个找到的因数一定是质数？

假设 i 是第一个能整除 n 的数（从 2 开始枚举）：

如果 i 是合数，则 i 有更小质因数 j < i

那么 j 也能整除 n，与 "i 是第一个" 矛盾

所以 i 必为质数 ✓
